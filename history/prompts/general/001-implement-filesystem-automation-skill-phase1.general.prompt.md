### Key Design Decisions

- **Fail-fast**: `SecurityError` raised immediately on boundary violations
- **Atomic ops**: `.bak` created before any mutation; restored on failure
- **Circuit Breaker**: 3-failure threshold, 5-min cooldown, HALF_OPEN probe
- **Idempotent**: `add_frontmatter` is a no-op if frontmatter already present
- **Dry-run**: Full operation simulation with zero filesystem side effects
